Verfügbarkeitsplanung

Hilfe > Verfügbarkeitsplanung

Übersicht

Die Verfügbarkeitsplanung ist das Herzstück der Mitarbeiterplanung. Hier legen Sie fest, wann welche Mitarbeiter verfügbar sind, und konfigurieren deren Arbeitszeiten, Standortpräferenzen und Fähigkeiten für jeden Tag der Planungsperiode.

Hauptfunktionen

  • Mitarbeiter-Auswahl - Wählen Sie den zu planenden Mitarbeiter aus der Liste
  • Verfügbarkeitsplanung - Markieren Sie verfügbare Tage und Zeiten
  • Standort-Konfiguration - Setzen Sie Präferenzen für Arbeitsstandorte
  • Skill-Management - Verwalten Sie Fähigkeiten für spezifische Tage
  • API-Integration - Importieren Sie Verfügbarkeiten aus externen Systemen

Verfügbarkeiten festlegen

Grundlegende Bedienung

  1. Wählen Sie einen Mitarbeiter aus der linken Tabelle
  2. Klicken Sie die gewünschten Tageszeiten an (z.B. Morgen, Nachmittag)
  3. Aktivierte Zeiten werden farblich hervorgehoben
  4. Konfigurieren Sie bei Bedarf spezielle Einstellungen über die Buttons unter dem Kalender

Tageszeit-Buttons

Die Tageszeit-Buttons repräsentieren die Standard-Arbeitszeiten:

  • Klicken - Verfügbarkeit für diese Tageszeit aktivieren/deaktivieren
  • Rechtsklick - Zeitraum für diese Tageszeit anpassen
  • Farbkodierung - Verschiedene Farben zeigen den Status an

💡 Tipp: Gruppenmodus

Verwenden Sie den Gruppenmodus um mehrere Tage gleichzeitig zu bearbeiten. Dies ist besonders praktisch für wiederkehrende Verfügbarkeitsmuster.

Erweiterte Konfiguration

Standort-Kombinationen

Legen Sie fest, an welchen Standort-Kombinationen der Mitarbeiter arbeiten kann:

  • Pro Tag - Klicken Sie auf den entsprechenden Button unter dem Kalender
  • Für alle Tage - Nutzen Sie "Location Combinations" im Seitenmenü
  • Reset - "Location Combinations → Reset" setzt auf Standardwerte zurück

Standort-Präferenzen

Bewerten Sie die Präferenz des Mitarbeiters für verschiedene Arbeitsstandorte:

  • Bewertungsskala - 1 (niedrig) bis 10 (hoch)
  • Standard - Wert 1 bedeutet neutrale/Standard-Präferenz
  • Pro Tag anpassen - Klicken Sie auf den entsprechenden Button
  • Reset - "Location Prefs → Reset" für Standardwerte

Partner/Standort-Präferenzen

Konfigurieren Sie Präferenzen für die Zusammenarbeit mit anderen Mitarbeitern an bestimmten Standorten:

  • Partner-Auswahl - Wählen Sie bevorzugte Arbeitspartner
  • Standort-spezifisch - Verschiedene Präferenzen je Standort
  • Bewertungssystem - Ähnlich wie Standort-Präferenzen

Skill-Management

Verwalten Sie die Fähigkeiten des Mitarbeiters für spezifische Tage:

  • Pro Tag - Klicken Sie auf den Skill-Button des entsprechenden Tages
  • Standard - Mitarbeiter startet mit seinen Standard-Fähigkeiten
  • Anpassen - Fügen Sie tagesspezifische Skills hinzu oder entfernen Sie welche
  • Reset alle - "Skills" Menü für globale Änderungen

Seitenmenü-Funktionen

Requested Assignments

Legen Sie fest, wie viele Einsätze der Mitarbeiter in dieser Periode haben soll:

  • Anzahl - Gewünschte Anzahl von Einsätzen
  • Required - Sind diese Einsätze verpflichtend?
  • Planungsrelevanz - Beeinflusst die automatische Planung

Times of Day

Verwalten Sie die Standard-Arbeitszeiten für diesen Mitarbeiter:

  • Neue Zeiten - Hinzufügen spezifischer Arbeitszeiten
  • Zeiten bearbeiten - Start- und Endzeiten anpassen
  • Entfernen - Nicht benötigte Zeiten löschen

API-Integration

Importieren Sie Verfügbarkeiten aus externen Systemen:

  • Einzelner Mitarbeiter - "Fetch for this employee"
  • Alle Mitarbeiter - "Fetch for all employees"
  • Überschreibung - Bestehende Daten werden ersetzt

Workflow-Beispiele

Neuen Mitarbeiter komplett einrichten

  1. Mitarbeiter aus der linken Liste auswählen
  2. Times of Day - Arbeitszeiten festlegen
  3. Requested Assignments - Gewünschte Einsätze definieren
  4. Verfügbarkeiten markieren - Tage anklicken
  5. Standort-Präferenzen setzen - Location Preferences konfigurieren
  6. Skills zuweisen - Fähigkeiten für spezielle Tage anpassen

Schnelle Verfügbarkeits-Planung

  1. Mitarbeiter auswählen
  2. API-Import nutzen - Falls externe Daten verfügbar
  3. Manuell nachbearbeiten - Spezielle Tage anpassen
  4. Gruppenmodus verwenden - Für wiederkehrende Muster

Bulk-Operationen

  1. Reset Time Input Field - Alle Zeiten auf Standard zurücksetzen
  2. Location Combinations → Reset - Alle Standort-Kombinationen zurücksetzen
  3. Skills → Reset Skills - Alle Skills auf Mitarbeiter-Standard
  4. Skills → Remove Skill - Alle Skills entfernen

Bedienelemente im Detail

Kalender-Ansicht

  • Monats-Header - Zeigt Monate der Planungsperiode
  • Tagesnummern - Datum des jeweiligen Tages
  • Wochentage - Wochenenden sind farblich hervorgehoben
  • Zeit-Buttons - Klickbar für Verfügbarkeits-Aktivierung
  • Konfigurations-Buttons - Unter jedem Tag für erweiterte Einstellungen

Konfigurations-Buttons (unter Kalender)

  • Location Combinations - Quadratischer Button für Standort-Kombinationen
  • Location Preferences - Button für Standort-Präferenzen
  • Partner/Location Prefs - Button für Partner-Standort-Präferenzen
  • Skills - Button für tagesspezifische Fähigkeiten

💡 Farbkodierung der Konfigurations-Buttons

  • Standard-Farbe - Keine Verfügbarkeit an diesem Tag
  • Grün - Einstellungen entsprechen den Standard-Werten
  • Gelb/Orange - Einstellungen weichen von Standard-Werten ab

Notizen und Informationen

Planungszeitraum-Notizen

Im unteren Bereich finden Sie zwei Textfelder:

  • Infos zum Planungszeitraum der Person - Spezifische Notizen für diese Planungsperiode
  • Infos zur Person - Allgemeine Notizen zum Mitarbeiter

Automatische Speicherung

Alle Änderungen in den Notiz-Feldern werden automatisch gespeichert, sobald Sie das Feld verlassen oder zu einem anderen Mitarbeiter wechseln.

Tastaturkürzel

  • F1 - Diese Hilfe öffnen
  • Tab - Zwischen Eingabefeldern wechseln
  • Enter - Verfügbarkeits-Button aktivieren (wenn fokussiert)
  • Rechtsklick - Kontextmenü für Zeit-Anpassungen

Häufige Probleme & Lösungen

Buttons sind deaktiviert

Problem: Zeit-Buttons oder Konfigurations-Buttons sind ausgegraut.

Ursache: Der Mitarbeiter ist an diesem Tag nicht dem entsprechenden Team zugeordnet.

Lösung: Prüfen Sie die Team-Zuordnung in den Team-Einstellungen.

Zeit-Standards fehlen

Problem: Fehlermeldung "No default time-of-day values are defined".

Lösung: Definieren Sie Standard-Arbeitszeiten über "Times of Day..." im Seitenmenü.

Gruppenmodus-Konflikte

Problem: Fehlermeldung beim Löschen von Verfügbarkeiten in Gruppen.

Lösung: Verwenden Sie den Gruppenmodus-Dialog zur Bereinigung der Gruppenstruktur.

API-Import schlägt fehl

Problem: Fehler beim Import von Verfügbarkeiten über die API.

Lösung: Prüfen Sie die Netzwerkverbindung und API-Konfiguration. Versuchen Sie einen manuellen Import.

Tipps für effiziente Planung

🚀 Best Practices

  • API zuerst - Nutzen Sie den API-Import für Basis-Verfügbarkeiten
  • Templates verwenden - Kopieren Sie Konfigurationen zwischen ähnlichen Mitarbeitern
  • Gruppenmodus nutzen - Für wiederkehrende Muster und Bulk-Operationen
  • Reset-Funktionen - Bei Fehlern auf Standardwerte zurücksetzen
  • Notizen pflegen - Dokumentieren Sie besondere Umstände in den Notiz-Feldern

⚡ Zeitersparende Tricks

  • Seitenmenü - Alle wichtigen Funktionen direkt zugänglich
  • Bulk-Reset - Nutzen Sie die Reset-Buttons für schnelle Korrekturen
  • Rechtsklick-Menüs - Schnelle Zeit-Anpassungen direkt am Button
  • Wochenend-Erkennung - Automatische Hervorhebung von Samstag/Sonntag

Verwandte Funktionen

Diese Funktionen arbeiten eng mit der Verfügbarkeitsplanung zusammen:

  • Gruppenmodus - Aufruf über "Switch to Group Mode"
  • Erweiterte Gruppeneigenschaften - Verfügbarkeits-Bedingungen
  • Team-Verwaltung - Team-Zuordnungen und Mitglieder
  • Stammdaten - Mitarbeiter-Grunddaten und Standorte
  • Plan-Ansicht - Ergebnis der Verfügbarkeitsplanung
  • Plan-Berechnung - Automatische Planung basierend auf Verfügbarkeiten
