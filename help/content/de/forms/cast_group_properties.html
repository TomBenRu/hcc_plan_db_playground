Cast-Gruppen-Eigenschaften

Hilfe > Cast-Gruppen-Eigenschaften

Übersicht

Der Cast-Gruppen-Eigenschaften-Dialog ermöglicht die detaillierte Konfiguration einzelner Cast-Gruppen. Sie können Fixed Cast definieren, Cast-Regeln festlegen, die Anzahl der Akteure bestimmen und die Striktheit der Regelbefolgung einstellen.

Hauptfunktionen

  • Fixed Cast Management: Definition fester Besetzungen für Gruppen
  • Cast-Regeln: Anwendung vorhandener oder benutzerdefinierter Casting-Regeln
  • Akteur-Anzahl: Festlegung der benötigten Anzahl von Darstellern
  • Regel-Striktheit: Kontrolle über die Flexibilität der Regelbefolgung
  • Child-Synchronisation: Übertragung von Eigenschaften auf Unter-Gruppen
  • Konsistenz-Warnungen: Automatische Erkennung von Konflikten

Grundbedienung

Fixed Cast bearbeiten

  1. Klicken Sie auf "Bearbeiten..." neben "Fixed Cast"
  2. Wählen Sie die gewünschten Akteure im Cast-Builder
  3. Bestätigen Sie die Auswahl
  4. Das System zeigt die Besetzung als Klartext an

Cast-Regeln konfigurieren

  1. Vorhandene Regel: Wählen Sie aus der Dropdown-Liste
  2. Benutzerdefiniert: Geben Sie eine eigene Regel ein
  3. Neue Regel: Erstellen Sie eine neue wiederverwendbare Regel
  4. Das System prüft automatisch auf Konflikte

Bedienelemente im Detail

Fixed Cast Bereich

  • Bearbeiten-Button: Öffnet den Cast-Builder für die Auswahl
  • Anzeige-Feld: Zeigt die aktuelle Besetzung in lesbarer Form
  • Warnung: Informiert über Konflikte mit Unter-Gruppen
  • Child-Korrektur: Dropdown-Menü für Synchronisation mit Unter-Gruppen

Cast-Regeln Bereich

  • Regel-Dropdown: Auswahl aus vorhandenen Cast-Regeln
  • Custom-Regel-Feld: Eingabe eigener Regeln (Großbuchstaben, Symbole: *, ~, -)
  • Neue Regel: Button zum Erstellen wiederverwendbarer Regeln
  • Regel-Warnung: Anzeige von Konflikten und Inkonsistenzen

Akteur-Anzahl Bereich

  • Spin-Box: Numerische Eingabe der benötigten Akteur-Anzahl
  • Warnung: Information über abweichende Child-Werte
  • Child-Korrektur: Button zur Angleichung aller Unter-Gruppen

Regel-Striktheit

  • Slider: 3 Stufen der Regel-Befolgung (0-2)
  • Text-Anzeige: Klare Beschreibung der aktuellen Einstellung
  • Auswirkung: Beeinflusst die Flexibilität beim Casting

Workflows

Cast-Gruppe vollständig konfigurieren

  1. Öffnen Sie die Eigenschaften der gewünschten Gruppe
  2. Fixed Cast: Definieren Sie feste Besetzungen falls gewünscht
  3. Cast-Regel: Wählen Sie eine passende Regel oder erstellen Sie eine neue
  4. Akteur-Anzahl: Stellen Sie die benötigte Anzahl ein
  5. Striktheit: Bestimmen Sie die Flexibilität der Regeln
  6. Prüfen Sie die Warnungen auf mögliche Konflikte
  7. Speichern Sie die Konfiguration

Hierarchie-Eigenschaften synchronisieren

  1. Konfigurieren Sie die Parent-Gruppe optimal
  2. Nutzen Sie die "Child-Korrektur" Buttons für:
    • Fixed Cast: "Untergeordnete Besetzungen löschen" oder "angleichen"
    • Akteur-Anzahl: "Anzahl der Child-Elemente korrigieren"
  3. Das System übernimmt die Werte für alle Unter-Gruppen

Tastaturkürzel

  • F1: Diese Hilfe öffnen
  • Tab: Navigation zwischen Eingabefeldern
  • Ctrl+S: Speichern
  • Esc: Dialog abbrechen
  • F2: Cast-Builder öffnen (wenn Fixed Cast fokussiert)

Fehlerbehebung

Cast-Regel-Konflikte

  • Fixed Cast Konflikt: Cast-Regel verhindert Fixed Cast - wählen Sie eines
  • Child-Konflikt: Unter-Gruppen haben abweichende Regeln
  • Lösung: Vereinheitlichen Sie die Regeln oder entfernen Sie Konflikte

Akteur-Anzahl-Warnungen

  • Orange Warnung: Unter-Gruppen haben unterschiedliche Anzahlen
  • Grün: Alle Werte sind konsistent
  • Korrektur: Nutzen Sie den "Child-Korrektur" Button

Fixed Cast Probleme

  • Zu wenige Akteure: Fixed Cast übersteigt verfügbare Anzahl
  • Regel-Konflikt: Cast-Regel verhindert Fixed Cast
  • Child-Abweichung: Unter-Gruppen haben andere Fixed Casts

Tipps & Best Practices

Cast-Regeln optimal nutzen

  • Wiederverwendung: Erstellen Sie Regeln für häufige Patterns
  • Hierarchie: Parent-Regeln sollten flexibler sein als Child-Regeln
  • Dokumentation: Verwenden Sie aussagekräftige Regel-Namen

Performance-Optimierung

  • Fixed Cast sparsam: Nur wo wirklich nötig definieren
  • Regel-Komplexität: Einfache Regeln sind performanter
  • Child-Synchronisation: Nutzen Sie Korrektur-Buttons für Batch-Updates

Verwandte Funktionen

  • Cast-Gruppen - Haupt-Management für Cast-Gruppen
  • Gruppen-Modus - Grundeinstellungen für Gruppierungen

Technische Details

  • Vererbung: Child-Properties überschreiben Parent-Properties
  • Validierung: Echtzeit-Prüfung aller Eingaben
  • Command-Pattern: Alle Änderungen über Undo-fähige Commands
  • Konsistenz-Engine: Automatische Konflikt-Erkennung

Support: Bei Problemen mit Cast-Gruppen-Eigenschaften überprüfen Sie zunächst die Konsistenz-Warnungen. Komplexe Regel-Konflikte können durch Vereinfachung der Hierarchie gelöst werden.
