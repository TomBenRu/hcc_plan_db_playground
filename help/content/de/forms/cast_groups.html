Cast-Gruppen

Hilfe > Cast-Gruppen

Übersicht

Der Cast-Gruppen-Dialog ermöglicht die hierarchische Verwaltung von Besetzungsgruppen. Sie können Gruppen erstellen, organisieren und deren Struktur per Drag & Drop anpassen. Das System unterstützt mehrere Ebenen von Unter-Gruppen für komplexe Cast-Strukturen.

Hauptfunktionen

  • Gruppen-Hierarchie: Tree-Ansicht aller Cast-Gruppen mit Parent-Child-Beziehungen
  • Drag & Drop: Intuitive Neuorganisation durch Ziehen von Gruppen
  • Gruppen-Erstellung: Schnelles Hinzufügen neuer Cast-Gruppen
  • Gruppen-Löschung: Entfernen nicht benötigter Gruppen
  • Eigenschaften-Bearbeitung: Direkter Zugriff auf Gruppen-Properties
  • Konsistenz-Prüfung: Automatische Validierung der Gruppen-Struktur

Grundbedienung

Neue Gruppe erstellen

  1. Klicken Sie auf "Neue Gruppe"
  2. Die neue Gruppe wird automatisch im Tree erstellt
  3. Die Gruppe kann sofort per Drag & Drop verschoben werden
  4. Doppelklick öffnet die Eigenschaften-Bearbeitung

Gruppen organisieren

  1. Drag & Drop: Ziehen Sie Gruppen an neue Positionen
  2. Hierarchie: Ziehen Sie Gruppen auf andere Gruppen für Parent-Child-Beziehungen
  3. Reihung: Ändern Sie die Reihenfolge durch horizontales Verschieben

Bedienelemente im Detail

Tree-Widget (Haupt-Ansicht)

  • Spalten: Titel, Location, Datum, Tageszeit, Anzahl Akteure, Fixed Cast, Regel, Strikt
  • Doppelklick: Öffnet Eigenschaften-Dialog
  • Auswahl: Einzelauswahl für Löschung und Bearbeitung
  • Expansion: Hierarchie-Ebenen ein-/ausklappen

Aktions-Buttons

  • Neue Gruppe: Erstellt eine neue Cast-Gruppe auf oberster Ebene
  • Gruppe entfernen: Löscht die ausgewählte Gruppe (nur wenn leer)
  • Speichern: Übernimmt alle Änderungen dauerhaft
  • Abbrechen: Verwirft alle Änderungen (Undo-System)

Workflows

Neue Cast-Struktur aufbauen

  1. Erstellen Sie Haupt-Gruppen mit "Neue Gruppe"
  2. Doppelklick auf Gruppe → Eigenschaften definieren
  3. Erstellen Sie Unter-Gruppen und ziehen Sie diese auf Parent-Gruppen
  4. Definieren Sie spezifische Fixed-Casts und Regeln je Ebene
  5. Testen Sie die Struktur mit Konsistenz-Prüfung

Bestehende Struktur anpassen

  1. Öffnen Sie die Cast-Gruppen-Ansicht
  2. Verwenden Sie Drag & Drop für strukturelle Änderungen
  3. Bearbeiten Sie Eigenschaften per Doppelklick
  4. Entfernen Sie überflüssige Gruppen
  5. System prüft automatisch auf Konsistenz

Tastaturkürzel

  • F1: Diese Hilfe öffnen
  • Enter: Eigenschaften-Dialog öffnen (wenn Gruppe ausgewählt)
  • Delete: Ausgewählte Gruppe entfernen
  • Ctrl+S: Speichern
  • Esc: Dialog abbrechen

Fehlerbehebung

Gruppe kann nicht gelöscht werden

  • Ursache: Gruppe enthält noch Unter-Gruppen oder Events
  • Lösung: Entfernen Sie zuerst alle Child-Elemente
  • Alternativ: Verschieben Sie Child-Elemente zu anderen Groups

Struktur-Inkonsistenzen

  • Solo-Children: Gruppen mit nur einem Child werden automatisch vereinfacht
  • Unused Groups: Leere Gruppen werden beim Speichern entfernt
  • Sichtbarkeits-Filter: System erweitert Ansicht bei Bedarf automatisch

Drag & Drop funktioniert nicht

  • Prüfung: Ist die Ziel-Position gültig?
  • Hierarchie: Events können nicht zu Groups mit anderen Events
  • Refresh: Bei Problemen speichern und Dialog neu öffnen

Tipps & Best Practices

Effizienz-Tipps

  • Batch-Bearbeitung: Definieren Sie Parent-Properties vor Child-Properties
  • Templates: Nutzen Sie bewährte Gruppen-Strukturen als Vorlage
  • Konsistenz: Prüfen Sie regelmäßig die Hierarchie-Logik

Strukturierungs-Empfehlungen

  • Logische Gruppierung: Orientieren Sie sich an dramaturgischen Einheiten
  • Flexibilität: Vermeiden Sie zu tiefe Hierarchien (max. 3-4 Ebenen)
  • Benennung: Verwenden Sie aussagekräftige Gruppen-Namen

Verwandte Funktionen

  • Gruppen-Eigenschaften - Detaillierte Einstellungen für Cast-Gruppen
  • Gruppen-Modus - Grundeinstellungen für Gruppierungen

Technische Details

  • Undo-System: Alle Aktionen sind rückgängig machbar
  • Auto-Speichern: Änderungen werden beim "Speichern" permanent
  • Konsistenz-Engine: Automatische Validierung der Gruppen-Logik
  • Event-Integration: Seamlose Verbindung mit Event-Management

Support: Für technische Probleme mit Cast-Gruppen wenden Sie sich an den Support oder überprüfen Sie die Gruppen-Konsistenz im Eigenschaften-Dialog.
