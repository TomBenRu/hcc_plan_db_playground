Availability Request Dialog

Hilfe > E-Mail-System > Availability Request

Übersicht

Der Availability Request Dialog sendet Anfragen zur Mitarbeiter-Verfügbarkeit für bestimmte Planungszeiträume. Dies ermöglicht eine effiziente Sammlung von Verfügbarkeitsdaten vor der Planerstellung.

Hauptfunktionen

  • Zeitraum-spezifische Anfragen - Verfügbarkeit für bestimmte Planungsperioden
  • Team-Integration - Anfragen an relevante Team-Mitglieder
  • Standard-Vorlagen - Vorgefertigte Verfügbarkeitsanfrage-Texte
  • Antwort-Integration - Verknüpfung mit Verfügbarkeits-System

Anwendungsbereich

  • Verfügbarkeitsabfragen - Ermittlung freier Mitarbeiter-Kapazitäten
  • Planungsvorbereitung - Sammlung von Verfügbarkeitsdaten vor Planerstellung
  • Urlaubsplanung - Abstimmung von Urlaubszeiten im Team
  • Kapazitätsplanung - Langfristige Personalplanung

Verwandte Funktionen

  • Custom Email Dialog - Personalisierte E-Mails
  • E-Mail-System Übersicht - Alle E-Mail-Funktionen
  • Verfügbarkeits-Verwaltung - Mitarbeiter-Verfügbarkeiten verwalten
