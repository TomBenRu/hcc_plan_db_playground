Event-Eigenschaften

Hilfe > Event-Eigenschaften

Übersicht

Der Event-Eigenschaften-Dialog ermöglicht die Bearbeitung von Einstellungen, die spezifisch für ein einzelnes Event gelten. Hier können Sie die feste Besetzung (Fixed Cast), die Anzahl der benötigten Mitarbeiter und Präferenz-Einstellungen konfigurieren.

Zugriff auf den Dialog

Sie erreichen diesen Dialog auf zwei Wegen:

  • Aus der Plan-Ansicht - Doppelklick auf einen Termin, dann "Event-Eigenschaften..." klicken
  • Aus den Cast-Gruppen - Doppelklick auf ein Event im Cast-Gruppen-Baum

Hauptfunktionen

Fixed Cast (Feste Besetzung)

Mit der festen Besetzung legen Sie fest, welche Mitarbeiter für dieses Event eingeteilt werden müssen oder können.

  • Bearbeiten-Button - Öffnet den Fixed Cast Builder für detaillierte Konfiguration
  • Anzeige - Zeigt die aktuelle Fixed Cast Konfiguration an

Weitere Informationen finden Sie unter Fixed Cast Builder.

Events mit Fixed Cast bevorzugen

Diese Checkbox ist nur aktiv, wenn:

  1. Ein Fixed Cast für das Event definiert ist
  2. Das Event zu einer Event-Gruppe gehört, bei der nicht alle Events gleichzeitig stattfinden

Wenn aktiviert, wird der Solver Events mit Fixed Cast gegenüber anderen Events bevorzugen.

Anzahl Mitarbeiter

Legt fest, wie viele Mitarbeiter für dieses Event benötigt werden.

  • Minimalwert - 0 (kein Mitarbeiter erforderlich)
  • Maximalwert - 99

Wichtiger Hinweis bei Reduzierung

Wenn Sie die Anzahl der Mitarbeiter reduzieren und bereits mehr Personen zugewiesen sind als die neue Anzahl erlaubt, werden Sie aufgefordert auszuwählen, welche Zuweisungen entfernt werden sollen.

Konsistenzprüfungen

Der Dialog führt automatische Prüfungen durch:

  • Fixed Cast vs. Anzahl Mitarbeiter - Die Anzahl der Mitarbeiter wird automatisch angepasst, wenn der Fixed Cast mehr Personen erfordert
  • Überzählige Zuweisungen - Bei Reduzierung der Mitarbeiterzahl werden Sie zur Bereinigung aufgefordert

Tastaturkürzel

  • F1 - Diese Hilfe öffnen
  • Enter - Speichern und schließen
  • Escape - Änderungen verwerfen und schließen

Verwandte Funktionen

  • Fixed Cast Builder - Detaillierte Konfiguration der festen Besetzung
  • Termin bearbeiten - Zuweisung von Mitarbeitern zu Terminen
  • Cast-Gruppen-Eigenschaften - Eigenschaften von Cast-Gruppen (mit zusätzlichen Optionen wie Cast-Regeln)

Unterschied zu Cast-Gruppen-Eigenschaften

Der Event-Eigenschaften-Dialog ist eine vereinfachte Version der Cast-Gruppen-Eigenschaften. Er enthält nur die für einzelne Events relevanten Einstellungen:

Einstellung Event-Eigenschaften Cast-Gruppen-Eigenschaften
Fixed Cast Ja Ja
Events mit Fixed Cast bevorzugen Ja Ja
Anzahl Mitarbeiter Ja Ja
Cast-Regeln Nein Ja
Strenge der Regelbefolgung Nein Ja
Kind-Elemente korrigieren Nein Ja
