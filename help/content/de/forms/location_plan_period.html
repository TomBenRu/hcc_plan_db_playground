Standort-Planungsperioden

Hilfe > Standort-Planung

Übersicht

Die Standort-Planungsperioden ermöglichen es Ihnen, Events und Termine für verschiedene Arbeitsstandorte zu planen. Hier legen Sie fest, wann an welchen Standorten Veranstaltungen stattfinden, wie viele Mitarbeiter benötigt werden und welche speziellen Anforderungen gelten.

Hauptfunktionen

  • Standort-Auswahl - Wählen Sie den zu planenden Standort aus der Liste
  • Event-Kalender - Planen Sie Events für spezifische Tage und Zeiten
  • Cast-Verwaltung - Bestimmen Sie die Anzahl und Art der benötigten Mitarbeiter
  • Skill-Management - Legen Sie erforderliche Fähigkeiten für Events fest
  • Notizen-System - Dokumentieren Sie wichtige Informationen zu Events
  • Planungsregeln - Automatisieren Sie die Event-Erstellung basierend auf Regeln

Event-Planung

Grundlegende Bedienung

  1. Wählen Sie einen Standort aus der linken Tabelle
  2. Klicken Sie die gewünschten Event-Zeiten an (z.B. Morgen, Nachmittag)
  3. Aktivierte Events werden farblich hervorgehoben
  4. Konfigurieren Sie bei Bedarf Cast, Skills und Notizen über die Konfigurations-Buttons

Event-Zeit-Buttons

Die Zeit-Buttons repräsentieren mögliche Event-Zeiten am Standort:

  • Klicken - Event für diese Zeit aktivieren/deaktivieren
  • Rechtsklick - Erweiterte Optionen (Zeit ändern, Skills, Cast, Flags, Notizen)
  • Mitarbeiter-Anzahl - Erscheint im Kontextmenü nach Aktivierung

💡 Tipp: Kontextmenü-Funktionen

Nach dem Rechtsklick auf einen aktiven Event-Button stehen folgende Optionen zur Verfügung:

  • Times of Day - Zeitraum für diesen Event anpassen
  • Employees - Anzahl benötigter Mitarbeiter festlegen
  • Skills - Erforderliche Fähigkeiten definieren
  • Fixed Cast - Spezifische Mitarbeiter zuweisen
  • Flags - Besondere Kennzeichnungen setzen
  • Notes - Notizen zum Event hinzufügen

Cast-Verwaltung

Mitarbeiter-Anzahl festlegen

Bestimmen Sie, wie viele Mitarbeiter für Events benötigt werden:

  • Standard-Anzahl - Wird für alle neuen Events verwendet
  • Event-spezifisch - Über Rechtsklick → "Employees" anpassbar
  • Seitenmenü - "Cast Size" für globale Einstellungen

Fixed Cast

Weisen Sie spezifische Mitarbeiter zu Events zu:

  • Pro Event - Rechtsklick → "Fixed Cast" für einzelne Events
  • Pro Tag - Button unter dem Kalender für alle Events eines Tages
  • Seitenmenü - "Fixed Cast" für planperioden-weite Einstellungen
  • Reset - "Cast → Reset" setzt auf Standard-Werte zurück

Cast Groups für Planungsperiode

Verwalten Sie Cast-Gruppen für die gesamte Planungsperiode:

  • Global bearbeiten - "Edit Cast and Cast Groups for Planning Period"
  • Gruppen-Modi - "Switch to Fixed Cast Group Mode" für Bulk-Operationen
  • Konsistenz - Automatische Synchronisation zwischen Events

Skill-Management

Skills pro Event

Definieren Sie erforderliche Fähigkeiten für Events:

  • Pro Event - Rechtsklick → "Skills" auf Event-Button
  • Pro Tag - Skill-Button unter dem Kalender für alle Events eines Tages
  • Standard-Skills - Basieren auf den Standard-Skills des Standorts

Bulk-Skill-Operationen

Verwalten Sie Skills für alle Events der Planungsperiode:

  • "Skills" Menü - Dropdown-Menü mit globalen Optionen
  • Remove Skills - Entfernt alle Skills von allen Events
  • Reset Skills - Setzt alle Skills auf Standort-Standard zurück

💡 Skill-Farbkodierung

  • Grün - Skills entsprechen Standort-Standard
  • Gelb/Orange - Skills weichen vom Standard ab
  • Grau - Keine Events an diesem Tag

Seitenmenü-Funktionen

Cast Size

Legen Sie die Standard-Anzahl von Mitarbeitern für Events fest:

  • Standard-Wert - Wird für alle neuen Events verwendet
  • Planungsperiode-weit - Gilt für alle Events an diesem Standort
  • Anpassbar - Einzelne Events können davon abweichen

Set Events According to Rules

Automatisieren Sie die Event-Erstellung basierend auf vordefinierten Regeln:

  • Planungsregeln - Definieren Sie wiederkehrende Muster
  • Zeiträume - Automatische Event-Erstellung für Perioden
  • Effizienz - Spart Zeit bei wiederkehrenden Events

Times of Day

Verwalten Sie die verfügbaren Arbeitszeiten für diesen Standort:

  • Neue Zeiten - Hinzufügen spezifischer Event-Zeiten
  • Zeiten bearbeiten - Start- und Endzeiten anpassen
  • Standard-Zeiten - Definieren Sie wiederverwendbare Zeitslots

Reset Time of Day Input Field

Setzen Sie alle Event-Zeiten auf die Standard-Werte der Planungsperiode zurück:

  • Bulk-Operation - Wirkt auf alle Events der Periode
  • Zeitanpassungen - Verwirft individuelle Zeit-Anpassungen
  • Standardisierung - Stellt Konsistenz wieder her

Notizen-System

Event-Notizen

Dokumentieren Sie wichtige Informationen zu Events:

  • Pro Event - Rechtsklick → "Notes" auf Event-Button
  • Pro Tag - Notes-Button unter dem Kalender
  • Konsistenz - Gleiche Notizen für alle Events eines Tages

Standort-Notizen

Im unteren Bereich finden Sie zwei Notiz-Bereiche:

  • Planungsperiode-Notizen - Spezifisch für diese Planungsperiode am Standort
  • Standort-Notizen - Allgemeine Informationen zum Standort
  • Automatische Speicherung - Änderungen werden sofort gespeichert

Gruppenmodi

Event Group Mode

Bearbeiten Sie mehrere Events gleichzeitig:

  • Aktivierung - "Switch to Group Mode" Button
  • Gruppen-Erstellung - Wählen Sie zusammengehörige Events
  • Bulk-Operationen - Änderungen wirken auf die gesamte Gruppe
  • Effizienz - Spart Zeit bei wiederkehrenden Mustern

Fixed Cast Group Mode

Verwalten Sie Cast-Zuweisungen in Gruppen:

  • Cast-Gruppen - Fassen Sie Events mit ähnlichem Cast zusammen
  • Mitarbeiter-Zuweisungen - Effiziente Verwaltung von Zuordnungen
  • Konsistenz - Gleiche Cast-Konfiguration für verwandte Events

Workflow-Beispiele

Neuen Standort komplett einrichten

  1. Standort aus der linken Liste auswählen
  2. Cast Size - Standard-Mitarbeiteranzahl festlegen
  3. Times of Day - Verfügbare Event-Zeiten definieren
  4. Events planen - Gewünschte Termine anklicken
  5. Skills konfigurieren - Erforderliche Fähigkeiten festlegen
  6. Fixed Cast - Bei Bedarf spezifische Mitarbeiter zuweisen

Automatisierte Event-Erstellung

  1. Standort auswählen
  2. "Set Events According to Rules" - Automatisierte Erstellung
  3. Planungsregeln definieren - Wiederkehrende Muster festlegen
  4. Bulk-Anpassungen - Nachträgliche Feinabstimmung über Gruppenmodi

Event-Konfiguration verfeinern

  1. Events aktivieren
  2. Rechtsklick-Optionen nutzen - Skills, Cast, Flags, Notizen
  3. Tagesweise Konfiguration - Buttons unter dem Kalender
  4. Konsistenz prüfen - Farbkodierung der Konfigurations-Buttons beachten

Group Mode (Event-Gruppen)

Bearbeiten Sie mehrere Events als Gruppe:

  • Aktivierung - "Switch to Group Mode" verwandelt Events in Gruppen
  • Gruppierung - Wählen Sie zusammengehörige Events aus
  • Bulk-Änderungen - Änderungen wirken auf alle Events der Gruppe
  • Nummerierung - Events werden mit Gruppennummern markiert

Fixed Cast Group Mode

Verwalten Sie Cast-Zuweisungen in Gruppen:

  • Cast-Gruppen - Fassen Sie Events mit gleichem Cast zusammen
  • Mitarbeiter-Pool - Effiziente Zuweisung von Mitarbeitern
  • Planungsoptimierung - Bessere Ressourcennutzung

Erweiterte Funktionen

Planungsregeln

Automatisieren Sie die Event-Erstellung:

  • Regel-basiert - Definieren Sie wiederkehrende Muster
  • Zeiträume - Legen Sie Gültigkeitsbereiche fest
  • Bedingungen - Berücksichtigen Sie spezielle Umstände
  • Effizienz - Reduziert manuellen Aufwand erheblich

Flags und Kennzeichnungen

Markieren Sie Events mit besonderen Eigenschaften:

  • Event-Typen - Kategorisieren Sie verschiedene Event-Arten
  • Prioritäten - Kennzeichnen Sie wichtige Events

Reset-Funktionen

Reset Time of Day Input Field

Setzen Sie alle Event-Zeiten zurück:

  • Standard-Zeiten - Verwendet die Standard-Zeiten der Planungsperiode
  • Alle Events - Wirkt auf alle Events des Standorts
  • Bereinigung - Entfernt ungenutzte Zeit-Konfigurationen

Cast → Reset

Setzen Sie Fixed Cast für alle Events zurück:

  • Standard-Cast - Verwendet den Standard-Cast der Planungsperiode
  • Konsistenz - Stellt einheitliche Cast-Konfiguration her
  • Bulk-Operation - Spart Zeit bei großen Änderungen

Skills-Reset

Zwei Optionen für Skill-Zurücksetzung:

  • Remove Skills - Entfernt alle Skills komplett
  • Reset Skills - Setzt auf Standort-Standard zurück
  • Bestätigung - Sicherheitsabfrage vor Ausführung

Tastaturkürzel

  • F1 - Diese Hilfe öffnen

  • Enter - Event-Button aktivieren (wenn fokussiert)
  • Rechtsklick - Kontextmenü für Event-Konfiguration

Häufige Probleme & Lösungen

Buttons sind deaktiviert

Problem: Event-Buttons oder Konfigurations-Buttons sind ausgegraut.

Ursache: Der Standort ist an diesem Tag nicht dem entsprechenden Team zugeordnet.

Lösung: Prüfen Sie die Team-Zuordnung in den Team-Einstellungen.

Zeit-Standards fehlen

Problem: Fehlermeldung "No time of day standards are defined".

Lösung: Definieren Sie Standard-Event-Zeiten über "Times of Day..." im Seitenmenü.

Cast-Konfiguration inkonsistent

Problem: Cast-Buttons zeigen unterschiedliche Farben obwohl gleiche Konfiguration gewünscht.

Lösung: Verwenden Sie "Cast → Reset" oder den Fixed Cast Group Mode für Konsistenz.

Gruppenmodus-Konflikte

Problem: Events können nicht korrekt gruppiert werden.

Lösung: Beenden Sie den Gruppenmodus mit Escape und starten Sie neu. Prüfen Sie, ob alle Events derselben Kategorie angehören.

Technische Details

Event-Datenstruktur

Events bestehen aus verschiedenen Komponenten:

  • Basis-Event - Datum, Zeit, Standort
  • Cast-Group - Mitarbeiter-Anforderungen und -Zuweisungen
  • Skill-Groups - Erforderliche Fähigkeiten
  • Flags - Besondere Kennzeichnungen
  • Notes - Dokumentation und Zusatzinformationen

Datensynchronisation

Das System sorgt automatisch für Konsistenz:

  • Signal-System - Automatische Updates zwischen Komponenten
  • Validierung - Prüfung auf Konsistenz bei Änderungen
  • Rollback - Automatische Wiederherstellung bei Fehlern
