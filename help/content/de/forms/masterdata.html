Stammdaten

Hilfe > Stammdaten

Übersicht

In den Stammdaten verwalten Sie alle grundlegenden Informationen für Ihr Planungssystem. Dies ist die zentrale Stelle für die Verwaltung von Mitarbeitern und Arbeitsstandorten.

Hauptbereiche

Die Stammdaten sind in zwei Hauptbereiche unterteilt:

  • Employees (Mitarbeiter) - Verwaltung aller Personen und deren Daten
  • Facilities (Einrichtungen) - Verwaltung aller Arbeitsstandorte und Adressen

Mitarbeiter-Management

Neuen Mitarbeiter anlegen

  1. Wechseln Sie zum Tab Employees
  2. Klicken Sie auf Create Person
  3. Füllen Sie die Personaldetails aus:
    • Vorname - Vorname des Mitarbeiters
    • Nachname - Nachname des Mitarbeiters
    • Benutzername - Eindeutiger Login-Name
    • E-Mail - Kontakt-E-Mail-Adresse
    • Telefon - Telefonnummer
    • Adresse - Wohnadresse
  4. Speichern Sie mit OK

Mitarbeiter aus Excel importieren

  1. Bereiten Sie eine Excel-Datei vor mit den Spalten:
    • First name (Vorname)
    • Last name (Nachname)
    • username (Benutzername)
  2. Klicken Sie auf Import People from XLSX
  3. Wählen Sie Ihre Excel-Datei aus
  4. Überprüfen Sie die Importvorschau
  5. Bestätigen Sie den Import

💡 Excel-Import Tipp

Stellen Sie sicher, dass die Spaltenüberschriften exakt "First name", "Last name" und "username" lauten. Weitere Spalten werden ignoriert.

Mitarbeiter bearbeiten

  1. Wählen Sie einen Mitarbeiter aus der Liste
  2. Doppelklicken Sie auf den Eintrag ODER
  3. Klicken Sie auf Edit Person
  4. Nehmen Sie die gewünschten Änderungen vor
  5. Speichern Sie mit OK

Mitarbeiter löschen

  1. Wählen Sie einen Mitarbeiter aus der Liste
  2. Klicken Sie auf Delete Person
  3. Bestätigen Sie die Löschung im Dialog

⚠️ Warnung beim Löschen

Das Löschen von Mitarbeitern kann sich auf bestehende Planungen auswirken. Prüfen Sie vorher, ob der Mitarbeiter in aktuellen Plänen verwendet wird.

Einrichtungen-Management

Neue Einrichtung anlegen

  1. Wechseln Sie zum Tab Facilities
  2. Klicken Sie auf Create Location
  3. Füllen Sie die Standortdetails aus:
    • Name - Bezeichnung der Einrichtung
    • Adresse - Vollständige Anschrift
    • Stadt - Ort der Einrichtung
    • Postleitzahl - PLZ
    • Kontaktdaten - Telefon/E-Mail der Einrichtung
  4. Speichern Sie mit OK

Einrichtung bearbeiten

  1. Wählen Sie eine Einrichtung aus der Liste
  2. Doppelklicken Sie auf den Eintrag ODER
  3. Klicken Sie auf Edit Location
  4. Nehmen Sie die gewünschten Änderungen vor
  5. Speichern Sie mit OK

Wichtige Bedienelemente

Mitarbeiter-Tab Funktionen

  • Create Person - Neuen Mitarbeiter anlegen
  • Import People from XLSX - Mitarbeiter aus Excel-Datei importieren
  • Edit Person - Ausgewählten Mitarbeiter bearbeiten
  • Delete Person - Ausgewählten Mitarbeiter löschen

Einrichtungen-Tab Funktionen

  • Create Location - Neue Einrichtung anlegen
  • Edit Location - Ausgewählte Einrichtung bearbeiten
  • Delete Location - Ausgewählte Einrichtung löschen
  • Location Settings - Erweiterte Standort-Konfiguration

Listen-Navigation

  • Sortierung - Klicken Sie auf Spaltenüberschriften zum Sortieren
  • Suche - Nutzen Sie die Suchfunktion über der Tabelle
  • Auswahl - Klicken Sie auf eine Zeile zur Auswahl
  • Bearbeitung - Doppelklick öffnet den Bearbeitungsdialog

Häufige Probleme und Lösungen

❓ Excel-Import funktioniert nicht

Lösung: Überprüfen Sie die Spaltenüberschriften. Sie müssen exakt "First name", "Last name" und "username" lauten (englisch).

❓ Mitarbeiter kann nicht gelöscht werden

Lösung: Der Mitarbeiter ist möglicherweise in aktiven Plänen zugewiesen. Entfernen Sie zuerst alle Planungszuweisungen.

❓ Doppelte Benutzernamen

Lösung: Benutzernamen müssen eindeutig sein. Verwenden Sie z.B. Kombinationen aus Vor- und Nachname oder fügen Sie Nummern hinzu.

❓ Einrichtung wird in Plänen nicht angezeigt

Lösung: Überprüfen Sie, ob die Einrichtung korrekt angelegt und aktiviert ist. Kontrollieren Sie auch die Adressdaten.

💡 Best Practice

Legen Sie zuerst alle Mitarbeiter und Einrichtungen an, bevor Sie mit der Planung beginnen. Dies erleichtert die spätere Terminzuordnung erheblich.

Datenqualität

Achten Sie auf vollständige und korrekte Daten:

  • Mitarbeiter: Vollständige Namen, eindeutige Benutzernamen, aktuelle Kontaktdaten
  • Einrichtungen: Korrekte Adressen, Postleitzahlen, Kontaktinformationen
  • Regelmäßige Pflege: Aktualisieren Sie Daten bei Änderungen sofort
