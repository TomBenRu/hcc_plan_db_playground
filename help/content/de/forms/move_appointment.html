Termin Verschieben

Hilfe > Termin Verschieben

Übersicht

Die Funktion "Termin Verschieben" ermöglicht es Ihnen, einen bestehenden Termin auf einen anderen Tag und/oder zu einer anderen Tageszeit zu verlegen. Dabei werden automatisch nur gültige Tage und verfügbare Zeiten für den jeweiligen Standort angezeigt.

Hauptfunktionen

  • Flexibles Datum-Management - Termine innerhalb der Planungsperiode auf beliebige Tage verlegen
  • Tageszeit-Auswahl - Wechsel zwischen verfügbaren Zeiten am gewählten Standort
  • Kalender-Interface - Intuitive Datumsauswahl über Kalender-Widget
  • Planungsperioden-Compliance - Automatische Begrenzung auf gültige Zeiträume
  • Standort-Integration - Nur für den Standort verfügbare Tageszeiten werden angezeigt
  • Termin-Kontext - Vollständige Anzeige der aktuellen Termin-Details

Grundbedienung

  1. Termin-Information überprüfen - Lesen Sie die angezeigten Details zum aktuellen Termin (Standort, Datum, Zeit)
  2. Neues Datum wählen:
    • Klicken Sie im Kalender auf das gewünschte Ziel-Datum
    • Nur Tage innerhalb der Planungsperiode sind auswählbar
    • Das aktuelle Datum ist vorausgewählt
  3. Neue Zeit festlegen:
    • Wählen Sie aus dem "Tageszeit"-Dropdown die gewünschte Zeit
    • Nur am Standort verfügbare Zeiten werden angezeigt
    • Die aktuelle Zeit ist vorausgewählt
  4. Verschiebung bestätigen - Klicken Sie auf "OK" um den Termin zu verschieben

Tastaturkürzel

F1 - Diese Hilfe öffnen

Enter - Termin verschieben und Dialog schließen

Escape - Dialog ohne Änderungen schließen

Pfeiltasten - Im Kalender navigieren

Alt + Pfeil runter - Tageszeit-Dropdown öffnen

Space - Datum im Kalender auswählen

Anwendungsbereich

🎯 Das Verschieben von Terminen wird verwendet für:

  • Terminkonflikt-Lösung - Überschneidende Termine auf andere Zeiten verlegen
  • Wetter-bedingte Anpassungen - Outdoor-Events bei schlechtem Wetter verschieben
  • Kapazitäts-Optimierung - Termine gleichmäßiger über die Woche verteilen
  • Kundenwünsche - Terminänderungen auf Kundenwunsch schnell umsetzen
  • Ressourcen-Management - Bessere Auslastung durch zeitliche Umverteilung

Praktische Anwendungsfälle

💡 Häufige Szenarien

Doppelbuchung auflösen: Termin auf freien Tag verschieben, wenn zwei Events zur gleichen Zeit geplant wurden.

Wetter-Anpassung: Outdoor-Termine bei schlechter Prognose auf besseren Tag verlegen.

Mitarbeiter-Verfügbarkeit: Termine anpassen wenn Schlüsselpersonal an anderem Tag verfügbar ist.

Kapazitäts-Balancing: Termine von überlasteten auf ruhigere Tage verschieben.

Häufige Probleme & Lösungen

Problem: Kalender zeigt keine auswählbaren Tage

Ursache: Planungsperiode ist sehr kurz oder bereits abgelaufen.

Lösung: Überprüfen Sie die Planungsperiode - eventuell muss eine neue Periode erstellt werden.

Problem: Tageszeit-Dropdown ist leer

Ursache: Für den Standort sind keine Tageszeiten konfiguriert.

Lösung: Konfigurieren Sie in der Standort-Verwaltung die verfügbaren Tageszeiten für diesen Ort.

Problem: Termin kann nicht verschoben werden (OK grau)

Ursache: Technische Validierung verhindert die Verschiebung.

Lösung: Wählen Sie ein anderes Datum/Zeit-Kombination oder prüfen Sie Terminkonflikt.

Problem: Gewünschtes Datum nicht anklickbar

Ursache: Datum liegt außerhalb der aktuellen Planungsperiode.

Lösung: Verschieben Sie nur innerhalb des sichtbaren, verfügbaren Zeitraums.

Best Practices

🚀 Effizienz-Tipps

Termine clustern: Verschieben Sie verwandte Termine auf denselben Tag für bessere Effizienz.

Vorlaufzeit beachten: Berücksichtigen Sie bei Verschiebungen die Kommunikation an Mitarbeiter und Kunden.

Verfügbarkeiten prüfen: Kontrollieren Sie nach der Verschiebung die Mitarbeiter-Verfügbarkeit am neuen Termin.

Kalender-Navigation: Nutzen Sie Pfeiltasten für schnelle Navigation zwischen Tagen.

Erweiterte Funktionen

Planungsperioden-Integration

Die Termin-Verschiebung ist intelligent in die Planungsperioden integriert:

  • Automatische Begrenzung - Min/Max-Datum entspricht der Planungsperiode des Termins
  • Gültigkeits-Prüfung - Nur innerhalb gültiger Zeiträume können Termine verschoben werden
  • Konsistenz-Sicherung - Termine bleiben immer innerhalb ihrer ursprünglichen Planungsperiode

Standort-spezifische Zeiten

Die Tageszeit-Auswahl berücksichtigt standort-spezifische Konfigurationen:

  • Location-Plan-Period Abhängigkeit - Nur für diesen Standort konfigurierte Zeiten
  • Zeit-Index Sortierung - Chronologische Anordnung der verfügbaren Zeitslots
  • Dynamische Filterung - Automatische Anpassung je nach gewähltem Standort

Kalender-Funktionalität

Das CalendarLocale Widget bietet erweiterte Datums-Features:

  • Deutsche Lokalisierung - Korrekte Wochentags- und Monats-Namen
  • Tastatur-Navigation - Vollständige Bedienung ohne Maus möglich
  • Visueller Status - Verfügbare/nicht verfügbare Tage werden unterschiedlich dargestellt
  • Datum-Konvertierung - Automatische Umrechnung zwischen QDate und Python datetime

Datenfluss

Nach der Bestätigung stehen die neuen Werte über Properties zur Verfügung:

  • new_date Property - Liefert das gewählte Datum als datetime.date
  • new_time_of_day Property - Liefert das komplette TimeOfDay-Schema-Objekt
  • Validierte Werte - Nur gültige Kombinationen können ausgewählt werden
