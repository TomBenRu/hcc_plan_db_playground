Skill-Management

Hilfe > Skill-Management

Übersicht

Das Skill-Management-System verwaltet individuelle Fähigkeiten von Personen in HCC Plan. Es besteht aus vier spezialisierten Dialogen für Projekt-Skills verwalten, einzelne Skills bearbeiten, Skills auswählen und Skills zuweisen.

🎯 Skills vs. Skill-Gruppen

Skills (diese Seite): Individuelle Fähigkeiten von Personen ("Person X kann schauspielern")
Skill-Gruppen: Personalanforderungen für Orte/Events ("Wir brauchen 3 Schauspieler")

Hauptfunktionen

  • Skill-Verwaltung - Erstellen, bearbeiten und löschen von Projekt-Skills
  • Skill-Bearbeitung - Einzelne Skills mit Name und Beschreibung editieren
  • Skill-Auswahl - Aus verfügbaren Projekt-Skills auswählen
  • Skill-Zuweisung - Skills zu Personen oder Verfügbarkeiten zuweisen
  • Soft-Delete System - Sichere Löschung mit Wiederherstellungs-Option
  • Reset-Funktion - Zurücksetzen auf Person-Standard-Skills (nur Verfügbarkeiten)

Die vier Dialog-Typen

1. Skill-Verwaltung (Haupt-Dialog)

Zweck: Zentrale Verwaltung aller Skills eines Projekts

  • Skills-Tabelle - Zeigt Name, Beschreibung und Löschstatus
  • Add Skill - Erstellt neue Skills über Bearbeitungs-Dialog
  • Edit Skill - Bearbeitet ausgewählte Skills
  • Delete Skill - Markiert Skills zur Löschung (reversibel)
  • Undelete Skill - Stellt gelöschte Skills wieder her
  • Show deleted skills - Checkbox zum Anzeigen gelöschter Skills

2. Skill-Bearbeitung (Einzelner Skill)

Zweck: Erstellen oder bearbeiten eines einzelnen Skills

  • Name-Feld - Kurze, prägnante Bezeichnung des Skills
  • Beschreibungs-Feld - Detaillierte Erklärung des Wissensgebiets
  • Dual-Modus - Funktioniert sowohl für neue als auch bestehende Skills

3. Skill-Auswahl (Projekt-Skills)

Zweck: Auswahl verfügbarer Skills für Zuweisungen

  • Projekt-Skills-Liste - Zeigt alle verfügbaren Skills des Projekts
  • Intelligente Filterung - Blendet bereits zugewiesene Skills aus
  • Sortierte Darstellung - Alphabetisch sortierte Skill-Liste
  • Nur-Lese-Modus - Reine Auswahl-Funktionalität

4. Skill-Zuweisung (Personen/Verfügbarkeiten)

Zweck: Verwaltung von Skills für Personen oder spezifische Verfügbarkeiten

  • Kontext-sensitive Beschreibung - Unterschiedlich für Personen vs. Verfügbarkeiten
  • Add Skill - Öffnet Skill-Auswahl-Dialog
  • Remove Skill - Entfernt Skills von Person/Verfügbarkeit
  • Reset Skills - Nur für Verfügbarkeiten: Zurück zu Person-Standard
  • Zwei Modi - PersonShow (allgemeine Skills) und AvailDayShow (tagesspezifisch)

Grundbedienung

Skills für ein Projekt verwalten

  1. Öffnen Sie den Skills-Hauptdialog
  2. Verwenden Sie Add Skill um neue Skills zu erstellen
  3. Doppelklicken Sie auf Skills oder nutzen Sie Edit Skill zum Bearbeiten
  4. Nutzen Sie Delete Skill für nicht mehr benötigte Skills
  5. Aktivieren Sie Show deleted skills um gelöschte Skills zu sehen

Skills einer Person zuweisen

  1. Öffnen Sie den Skill-Zuweisungs-Dialog für die gewünschte Person
  2. Klicken Sie Add Skill um verfügbare Projekt-Skills anzuzeigen
  3. Wählen Sie den gewünschten Skill aus der Liste
  4. Bestätigen Sie mit OK
  5. Der Skill erscheint in der Person-Skills-Liste

Tagesspezifische Skills verwalten

  1. Öffnen Sie den Skill-Dialog für eine spezifische Verfügbarkeit
  2. Die Liste zeigt zunächst die Standard-Skills der Person
  3. Entfernen Sie nicht verfügbare Skills mit Remove Skill
  4. Fügen Sie zusätzliche Skills mit Add Skill hinzu
  5. Nutzen Sie Reset Skills um zu Person-Standard zurückzukehren

Praktische Anwendungsfälle

🎯 Szenario 1: Neues Projekt - Skill-Katalog erstellen

Dialog: Skill-Verwaltung. Definieren Sie alle Skills wie "Schauspiel", "Gesang", "Tanz", "Bühnentechnik". Jeder Skill erhält eine detaillierte Beschreibung der Anforderungen.

🎯 Szenario 2: Mitarbeiter einarbeiten - Skills zuweisen

Dialog: Skill-Zuweisung (Person). Neue Teammitglieder erhalten ihre Qualifikationen über den Zuweisungs-Dialog. Das System öffnet automatisch die Skill-Auswahl.

🎯 Szenario 3: Krankheitsfall - Tagesspezifische Anpassung

Dialog: Skill-Zuweisung (Verfügbarkeit). Ein Sänger hat Stimmprobleme und der "Gesang"-Skill wird für betroffene Tage entfernt. Reset-Funktion stellt später den Normalzustand wieder her.

🎯 Szenario 4: Skill-Archivierung

Dialog: Skill-Verwaltung. Nicht mehr benötigte Skills werden gelöscht, bleiben aber über "Show deleted skills" verfügbar und können wiederhergestellt werden.

Häufige Probleme & Lösungen

Problem: Skill kann nicht endgültig gelöscht werden

Ursache: Der Skill wird noch von Personen oder Verfügbarkeiten verwendet

Lösung: Entfernen Sie zuerst alle Zuweisungen über die Skill-Zuweisungs-Dialoge, dann erst endgültig löschen

Problem: Skill erscheint nicht in Auswahl-Liste

Ursache: Skill ist bereits zugewiesen und wird daher ausgeblendet

Lösung: Das ist normale Funktionalität - nur noch nicht zugewiesene Skills werden angezeigt

Problem: Reset Skills funktioniert nicht

Ursache: Reset-Funktion ist nur in Verfügbarkeits-Dialogen verfügbar, nicht bei Personen

Lösung: Für Person-Skills nutzen Sie Remove/Add Skills manuell

Best Practices

💡 Dialog-Workflow verstehen

Verwaltung → Bearbeitung → Auswahl → Zuweisung: Erstellen Sie Skills zentral, bearbeiten Sie sie einzeln, wählen Sie aus verfügbaren aus und weisen Sie sie spezifisch zu.

🚀 Skill-Hierarchie planen

Definieren Sie im Verwaltungs-Dialog zunächst alle benötigten Skills, bevor Sie mit Zuweisungen beginnen. Eine gute Skill-Basis erleichtert die spätere Zuordnung.

⚡ Verfügbarkeits-Skills clever nutzen

Nutzen Sie tagesspezifische Skills für Ausnahmen (Krankheit, fehlende Ausrüstung). Die Reset-Funktion macht Änderungen schnell rückgängig.

🔄 Soft-Delete für Flexibilität

Löschen Sie Skills zunächst nur zur Markierung. Endgültige Löschung erst nach Sicherstellung, dass keine Abhängigkeiten bestehen.

Dialog-spezifische Bedienung

Skill-Verwaltung (Projekt-Ebene)

  • Tabelle: 3 Spalten (Name, Beschreibung, Gelöscht-Status)
  • Doppelklick: Öffnet Bearbeitungs-Dialog
  • Show deleted skills: Checkbox für Anzeige gelöschter Skills
  • 4 Action-Buttons: Add, Edit, Delete, Undelete

Skill-Bearbeitung (Einzelner Skill)

  • Name-Feld: Pflichtfeld für Skill-Bezeichnung
  • Beschreibungs-Feld: Optionale Details zum Skill
  • Adaptive Überschrift: "New Skill" oder "Edit Skill"

Skill-Auswahl (Projekt-Skills)

  • Readonly-Tabelle: 2 Spalten (Name, Beschreibung)
  • Intelligente Filterung: Bereits zugewiesene Skills ausgeblendet
  • Einfache Auswahl: Klick + OK für Auswahl

Skill-Zuweisung (Personen/Verfügbarkeiten)

  • Kontext-Anzeige: Unterschiedliche Beschreibung für Personen vs. Verfügbarkeiten
  • Zugewiesene Skills: Tabelle der aktuell zugewiesenen Skills
  • Add Skill: Öffnet Skill-Auswahl-Dialog
  • Remove Skill: Entfernt Zuweisung
  • Reset Skills: Nur bei Verfügbarkeiten - zurück zu Person-Standard

Erweiterte Funktionen

Zweistufiger Löschprozess

Skills werden zunächst zur Löschung markiert (PrepDelete) und können über Undelete wiederhergestellt werden. Die endgültige Löschung erfolgt erst nach Bestätigung und nur bei nicht verwendeten Skills.

Verfügbarkeits-spezifische Features

Bei Verfügbarkeits-Skills steht zusätzlich Reset Skills zur Verfügung. Diese Funktion entfernt alle tagesspezifischen Skills und lädt die Standard-Skills der Person neu.

Intelligente Skill-Filterung

Der Skill-Auswahl-Dialog zeigt nur Skills an, die noch nicht zugewiesen sind. Dies verhindert Doppel-Zuweisungen und vereinfacht die Auswahl.

Häufige Workflows

🔄 Workflow 1: Neues Projekt Setup

  1. Skill-Verwaltung öffnen und alle benötigten Projekt-Skills definieren
  2. Für jeden Skill Add Skill → Name und Beschreibung eingeben
  3. Bei den Stammdaten Skills den Personen zuweisen

🔄 Workflow 2: Person-Skills anpassen

  1. Person in Stammdaten auswählen → Skills öffnen
  2. Add Skill → Aus verfügbaren Projekt-Skills auswählen
  3. Nicht zutreffende Skills mit Remove Skill entfernen

🔄 Workflow 3: Tagesspezifische Anpassungen

  1. Verfügbarkeits-Dialog für spezifischen Tag öffnen
  2. Bei Problemen: Remove Skill für betroffene Skills
  3. Bei zusätzlichen Fähigkeiten: Add Skill für temporäre Skills
  4. Bei Unsicherheit: Reset Skills für Neustart

Tastaturkürzel

  • F1 - Diese Hilfe öffnen (in allen Skill-Dialogen)
  • Doppelklick - Skill bearbeiten (in Verwaltungs-Dialog)
  • Enter - Aktion bestätigen / Dialog schließen
  • Escape - Dialog ohne Änderungen schließen
  • Tab - Zwischen Formularfeldern wechseln
  • Del - Ausgewählten Skill löschen (in Verwaltungs-Dialog)
  • Space - Checkbox umschalten (Show deleted skills)
