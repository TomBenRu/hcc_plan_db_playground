Systemanforderungen

HCC Plan benötigt folgende Komponenten:

  • Windows 10 oder höher
  • Python 3.11 oder höher
  • PySide6 Framework

Installation

  1. Laden Sie das Installationspaket herunter
  2. Führen Sie die setup.exe aus
  3. Folgen Sie dem Installations-Assistenten
  4. Starten Sie HCC Plan erstmalig
